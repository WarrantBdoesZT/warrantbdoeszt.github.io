ZTAuditor

Identifies network segmentation gaps and lateral movement paths attackers could exploit across your environment.

📘 How to Use

1 · Prerequisites

  • Windows 10 1809+ or Server 2019+ (domain-joined or with network access to the target domain)
  • .NET 8 SDK — download from dot.net/8 (SDK, not Runtime)
  • A standard Domain User account — no Domain Admin rights required
  • Remote Registry service running on target hosts (for registry-based checks; deploy via GPO)

2 · Run the executable

Build from source or grab a pre-built release, then run from a PowerShell prompt:

# Basic run — comma-separated hosts
.\ZeroTrustAuditor.exe --hosts DC01,SRV01,SRV02 --domain corp.local

# Using a targets file (one hostname per line)
.\ZeroTrustAuditor.exe --hosts-file .\targets.txt --domain corp.local

Common flags:

FlagDescription
--hosts h1,h2Comma-separated list of hostnames
--hosts-file file.txtPath to text file with one hostname per line
--domain corp.localActive Directory domain FQDN (required)
--output ./reportsOutput directory (default: ./reports)
--skip-modules A,BSkip modules: AdAuditor, ProtocolProbe, LateralPathAnalyzer, ShareAuditor, SegmentationChecker

3 · Read the output

Five audit modules run in parallel, then findings are deduplicated, scored, and boosted by correlation rules. Reports are written to the --output directory, timestamped in the filename:

FormatUse case
HTMLHuman-readable report — open in any browser
JSONMachine-readable, API ingest, custom tooling
CSVTicket creation in ServiceNow / Jira
Splunk HECPush to Splunk HTTP Event Collector
SentinelIngest to Microsoft Sentinel Log Analytics
CEFSyslog forwarding to ArcSight, QRadar, or any CEF collector

Each finding carries a severity and risk score — prioritize by severity:

SeverityScoreRecommended SLA
Critical9.024–48 hours
High7.07 days
Medium5.030 days
Low3.090 days

When two dangerous misconfigs appear on the same host, both scores get a +2.0 boost — the combination forms a complete attack chain.
